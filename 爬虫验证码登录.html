爬虫验证码登录

发表于 2018-08-22 | 分类于 爬虫 | 浏览 次

带验证码爬虫登录

随着爬虫技术的广泛应用，很多的网站也作了应对，比如最常见的令人头疼的字符验证码

我的思路是：使用selenium+phantomjs(chrome亦可)模拟登录，验证码通过截图获取，再进行验证码识别，识别后填充输入框，从而达到模拟登录的效果

字符验证码识别

简单字符

此类验证码可以通过OCR识别

java可以接第三方OCR服务，python也有开源的库pytesseract，其核心都是OCR

复杂字符

此类验证码干扰较多，ocr识别率较低，即便使用机器学习训练识别率也不会有太大提升，最好的方式是接第三方打码平台

打码平台接入

我选择的是 “人工”智能打码（即平台另一方是真人识别） 云打码

接口选择

考虑到多平台通用需求，则放弃了java sdk方式的接入，选择了web http接口

验证流程

1.云打码平台用户调用接口上传待验证验证码
2.调用获取验证码结果接口（时间不定，可能几秒也可能二十多秒，取决于验证码复杂度）

验证代码

String url = "http://api.yundama.com/api.php?method=result";
StringBody name = new StringBody(username, ContentType.MULTIPART_FORM_DATA);
StringBody pwd = new StringBody(password, ContentType.MULTIPART_FORM_DATA);
StringBody id = new StringBody(String.valueOf(appid), ContentType.MULTIPART_FORM_DATA);
StringBody key = new StringBody(appkey, ContentType.MULTIPART_FORM_DATA);
StringBody cid1 = new StringBody(cid, ContentType.MULTIPART_FORM_DATA);
HttpPost request = new HttpPost(url);
HttpEntity httpEntity = MultipartEntityBuilder.create().setMode(HttpMultipartMode.BROWSER_COMPATIBLE)
        .addPart("username",name)
        .addPart("password",pwd)
        .addPart("appid",id)
        .addPart("appkey",key)
        .addPart("cid",cid1)
        .build();
request.setEntity(httpEntity);
CookieStore cookieStore = (CookieStore) AuthUtil.getUsercookiemap().get(username);
String content = "";
String text = "";
try {
    //根据以往验证码识别时间来设置，在云打码平台用户和开发者用户均可看到
    Thread.sleep(12000);
    CloseableHttpResponse response = null;
    response = getResponse(request, cookieStore);
    content = EntityUtils.toString(response.getEntity(),"utf-8");
    JSON json = JSON.parseObject(content);
    log.info("result: ",((JSONObject) json).getString("text"));
    //{"ret":-3002,"cid":1776520707,"text":""} 此时验证码识别中，需重复调用
    if (((JSONObject) json).get("ret").equals(Integer.parseInt("-3002"))){
        response = getResponse(request, cookieStore);
        content = EntityUtils.toString(response.getEntity(),"utf-8");
        json = JSON.parseObject(content);
        log.info("result: ",((JSONObject) json).getString("text"));
        text = ((JSONObject) json).getString("text");
    }else {
        text = ((JSONObject) json).getString("text");
    }
    return text;
} catch (IOException e) {
    log.error("验证码识别","response error");
}

获取验证码截图

根据xpath进行元素截图代码

/**
 * 根据坐标位置截取验证码区域（x,y,w,h需微调）
 * @param element
 * @return
 * @throws Exception
 */
public static File captureElement(WebElement element) throws Exception {
    // 截图整个页面
    WrapsDriver wrapsDriver = (WrapsDriver)element;
    File screen = ((TakesScreenshot)wrapsDriver.getWrappedDriver()).getScreenshotAs(OutputType.FILE);
    BufferedImage img = ImageIO.read(screen);
    // 获得元素的高度和宽度
    int width = element.getSize().getWidth();
    int height = element.getSize().getHeight();
    // 创建一个矩形使用上面的高度，和宽度
    Rectangle rect = new Rectangle(width, height);
    // 得到元素的坐标
    Point p = element.getLocation();
    BufferedImage dest = img.getSubimage(p.getX()+25, p.getY(), rect.width, rect.height);
    // 存为png格式
    ImageIO.write(dest, "png", screen);
    return screen;
}
